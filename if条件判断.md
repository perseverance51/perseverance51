####  言归正传，步入主题。
1，if条件语句
```cpp
if...else...
if（条件）{
	满足条件时输出的内容
}else{
	内容
}
```
在if条件里面的做判断逻辑数据类型：整数型和字符串型的逻辑判断。
条件里不管是什么类型的数据，它们的所使用的逻辑都一样的。
#### 1.整形数据类型
逻辑运算符
&& 表达式：条件一 && 条件二 ，说明：仅仅两个条件同时为真，结果为真。
|| 表达式： 条件一 || 条件二， 说明：只要两个条件有一个为真，结果为真。
！表达式： ！条件， 说明：条件为真时，他为假；条件为假时，他为真。
